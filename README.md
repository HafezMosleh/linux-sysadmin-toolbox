## 🌟 Overview
A Swiss-army knife for server management. This collection of Bash scripts automates daily sysadmin tasks, from checking heavy RAM/CPU consuming processes to clearing page caches and analyzing disk usage anomalies.

## ✨ Key Features
- 📊 **Resource Monitoring**: Quickly spot top CPU/RAM hogs.
- 🧹 **System Maintenance**: Safe one-liners to clear caches.
- 🛠️ **DevOps Friendly**: Easily integrate into cron jobs for automation.

## 🚀 Getting Started

### Prerequisites
Make sure you have `Bash` installed on your local machine.

### Installation
```bash
git clone https://github.com/HafezMosleh/linux-sysadmin-toolbox.git
cd linux-sysadmin-toolbox
chmod +x main.sh
```

### Usage
```bash
./main.sh
```
